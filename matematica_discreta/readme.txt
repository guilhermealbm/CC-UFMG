Versão do GCC utilizada para testes: 7.3.0

Para executar o programa no gnu-linux/mac, siga os seguintes passos:

1 - Abra o terminal (ctrl + alt + t)
2 - Navegue até a pasta em que se encontra o arquivo .c (utilize os comandos cd e ls)
3 - Compile o programa uzando o GNU Compiler Collection com o seguinte comando: gcc rb.c
4 - Execute o arquivo gerado passando como parâmetro o arquivo de testes com o seguinte comando: ./a.out <teste.txt

Para executar o programa no Windows, siga os seguintes passos:

1 - Aperte Windows + R, digite "cmd" e aperte enter
2 - Navegue até a pasta em que se encontra o arquivo .c (utilize o comando cd)
3 - Compile o programa uzando o GNU Compiler Collection com o seguinte comando: gcc rb.c
4 - Execute o arquivo gerado passando como parâmetro o arquivo de testes com o seguinte comando: a.exe <teste.txt
